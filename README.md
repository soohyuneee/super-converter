- 기능소개

1. 단위 변환

   </br> 변환된 단위는 소수점 두 자리까지 보여집니다.

2. 뒤집기 & 초기화
   </br> 변환 순서를 뒤집을 수 있습니다.
   </br> 초기화를 할 수 있습니다.
